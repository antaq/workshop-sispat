Processo de integração massiva de dados

INTEGRAÇÃO

Ambiente de Produção

Ambiente de Teste / Homologação

https://des-apex.antaq.gov.br/ords/sfc_sispat/sispat/bens/

1

Arquivo JSON

A API recebe os dados no formato JSON. Prepare o payload com a estrutura exigida pela documentação (campos dos bens a cadastrar).

payload.json

[ { "cnpj": 049***08, "numero_registro_patrimonial": "PAT-2024-0001", "numero_do_contrato": "CT-2024-001", "nome_item": "Guindaste portuário", ... }, ... ]

2

Inserção

Envie o JSON para o endpoint da API (requisição POST) e valide o retorno. O sistema processa os dados e retorna a confirmação do cadastro.

POST Request

Status: 201 Bem atualizado com sucesso. { "message": "Operação realizada com sucesso.", "action": "Atualizado", "data": { "cnpj": "049***08", "numero_registro_patrimonial": "CT-2024-0001", "nome_item": "Guindaste portuário" } }
